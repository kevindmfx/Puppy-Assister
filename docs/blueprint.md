# **App Name**: Vision Weaver

## Core Features:

- Text Prompt Input: Allows the user to input a text prompt describing the desired image.
- Environment Selection: Provides a dropdown to select the environment type (e.g., indoor, outdoor, urban, natural).
- Lighting Selection: Provides a dropdown to select the lighting style (e.g., warm, cold, studio, natural).
- Subject Selection: Offers a dropdown menu for selecting the subject or object featured in the scene (person, landscape, vehicle, etc.)
- Mood Selection: Lets the user pick a feeling or atmosphere through a dropdown, such as calm, energetic, or mysterious.
- Image Customization: Offers dropdowns or input fields for determining image dimensions, level of detail, texture emphasis and camera angle.
- JSON Export: Generates and downloads a JSON file containing all user selections, designed for AI image prompt ingestion.

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) to evoke creativity and openness.
- Background color: Light, desaturated gray (#F0F0F0) for a clean and professional look.
- Accent color: Gentle lavender (#E6E6FA) for highlights and interactive elements.
- Headline font: 'Space Grotesk', sans-serif, for headlines. Body font: 'Inter', sans-serif, for text. Pairing of the modern grotesk-style 'Inter' with 'Space Grotesk' will lend a technical yet still accessible feel.
- A clean, modern layout with a header and footer. Input fields and dropdowns should be logically organized for easy use.
- Simple, intuitive icons to represent different image aspects and settings.
- Subtle, unobtrusive animations to enhance user interaction.
